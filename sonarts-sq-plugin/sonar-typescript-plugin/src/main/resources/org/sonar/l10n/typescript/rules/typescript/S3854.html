super() should only be invoked once in a constructor. Additional invocations will result in runtime errors.

Noncompliant Code Example

class Dog extends Animal {
  constructor(name) {
    super();
    this.name = name;
    super();         // Noncompliant
    super.doSomething();
  }
}

Compliant Solution

class Dog extends Animal {
  constructor(name) {
    super();
    this.name = name;
    super.doSomething();
  }
}
