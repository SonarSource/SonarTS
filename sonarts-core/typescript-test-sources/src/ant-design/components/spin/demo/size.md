## zh-CN

小的用于文本加载，默认用于卡片容器级加载，大的用于**页面级**加载。

## en-US

A small `Spin` use in loading text, default `Spin` use in loading card-level block, and large `Spin` use in loading **page**.

````jsx
import { Spin } from 'antd';

ReactDOM.render(
  <div>
    <Spin size="small" />
    <Spin />
    <Spin size="large" />
  </div>
, mountNode);
````

.ant-spin {
  margin-right: 16px;
}
</style>
